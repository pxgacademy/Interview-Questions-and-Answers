### _#49._ üü£ **What is Redux and how does it work with React?**

Redux ‡¶π‡¶≤‡ßã ‡¶è‡¶ï‡¶ü‡¶ø open-source JavaScript library ‡¶Ø‡¶æ predictable state management-‡¶è‡¶∞ ‡¶ú‡¶®‡ßç‡¶Ø ‡¶¨‡ßç‡¶Ø‡¶¨‡¶π‡ßÉ‡¶§ ‡¶π‡¶Ø‡¶º‡•§ React-‡¶è‡¶∞ ‡¶∏‡¶æ‡¶•‡ßá Redux global state manage ‡¶ï‡¶∞‡¶§‡ßá ‡¶¨‡ßç‡¶Ø‡¶¨‡¶π‡ßÉ‡¶§ ‡¶π‡¶Ø‡¶º, ‡¶Ø‡¶æ multiple components-‡¶è share ‡¶ï‡¶∞‡¶æ ‡¶Ø‡¶æ‡¶Ø‡¶º‡•§

- **Store:** Single source of truth, ‡¶Ø‡ßá‡¶ñ‡¶æ‡¶®‡ßá app-‡¶è‡¶∞ ‡¶∏‡¶¨ state store ‡¶π‡¶Ø‡¶º‡•§
- **Actions:** State change-‡¶è‡¶∞ ‡¶ú‡¶®‡ßç‡¶Ø payloads (objects) ‡¶Ø‡¶æ dispatch ‡¶ï‡¶∞‡¶æ ‡¶π‡¶Ø‡¶º‡•§
- **Reducers:** Pure functions ‡¶Ø‡¶æ actions-‡¶è‡¶∞ ‡¶≠‡¶ø‡¶§‡ßç‡¶§‡¶ø‡¶§‡ßá state update ‡¶ï‡¶∞‡ßá‡•§
- **React-Redux:** `Provider` ‡¶è‡¶¨‡¶Ç `useSelector`/`useDispatch` hooks ‡¶¶‡¶ø‡¶Ø‡¶º‡ßá Redux store-‡¶ï‡ßá React-‡¶è‡¶∞ ‡¶∏‡¶æ‡¶•‡ßá connect ‡¶ï‡¶∞‡ßá‡•§
